## üì¶ Installation

### Method 1: VS Code Marketplace (Recommended)

1. Open **VS Code**
2. Press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (macOS) to open Extensions
3. Search for **`Nuclear Hippie`**
4. Click **Install**
5. Activate the theme:
   - Press `Ctrl+K Ctrl+T` (Windows/Linux) or `Cmd+K Cmd+T` (macOS)
   - Select **Nuclear** or **Nuclear Pastele**

### Method 2: Manual Installation (.vsix)

1. Download the latest `.vsix` file from [Releases](https://github.com/mcagampan/nuclear-hippie/releases)
2. Open VS Code
3. Press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (macOS)
4. Click the `‚ãØ` (Views and More Actions) menu
5. Select **Install from VSIX...**
6. Choose the downloaded `.vsix` file
7. Activate the theme as described above

## üõ†Ô∏è Building from Source

Want to customize the theme or contribute? Follow these steps:

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- [vsce](https://github.com/microsoft/vscode-vsce) (VS Code Extension Manager)

### Build Steps

```bash
git clone https://github.com/mcagampan/nuclear-hippie.git
cd nuclear-hippie
npm install -g @vscode/vsce
vsce package
```

### Installing Your Custom Build

After building, install the generated `.vsix` file using **Method 2** above.

### Development Workflow

1. Make your changes to theme files in `themes/` directory
2. Press `F5` in VS Code to open a new Extension Development Host window
3. Test your changes in real-time
4. Package and install when satisfied
